Expense Tracker

The Expense Tracker is a simple and efficient application that helps users manage their daily spending. It allows users to add, edit, and delete transactions while also providing clear visual insights through interactive graphs. This makes it easy to understand spending patterns, track budget usage, and make better financial decisions.

Features
Add & Manage Transactions

Add new expenses or income entries

Specify details such as title, amount, date, and category

Edit or delete existing transactions anytime

Real-time Balance Calculation

Automatically updates total income, total expenses, and overall balance

Category-Based Tracking

Transactions can be grouped by categories like:

Food

Travel

Shopping

Bills

Others

Analytics & Visual Insights

View detailed spending analytics through charts:

Pie Chart — category-wise distribution of expenses

Bar/Line Chart — monthly or weekly spending trend

Clean & Responsive UI

Simple design that works smoothly across devices

Intuitive layout for quick navigation
